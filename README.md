# 再啦讚！

> 「在等我一下」
> 「再啦讚！」
>
> 「因該沒問題吧？」
> 「應啦讚！」

用了這個擴充功能，再也不會看到讓你暴走的錯字。

Chrome 擴充功能，自動修正常見中文打字錯誤：因該→應該、在/再判斷、的/得、券/卷、已/以、即/既、哪/那、做/作、帶/戴、座/坐、訂/定等 14 類。

詞組資料庫：**2395 組唯一詞組 + 15 條正則規則**，涵蓋 **14 類**常見錯字，等效覆蓋率 ~97%+。

## 功能

### 1. 固定修正
- **因該** → **應該**
- **再啦讚** → **在啦讚**（梗名保護）

### 2. 智能語境判斷（在 / 再）

| 錯誤用法 | 正確用法 | 說明 |
|---------|---------|------|
| 在見 | 再見 | 再次見面 |
| 在來 | 再來 | 再次前來 |
| 正再吃飯 | 正在吃飯 | 進行中的動作 |
| 再家 | 在家 | 位置所在 |
| 現再 | 現在 | 此時此刻 |

**判斷原則：**
- **再** = 重複、再次、然後（Again）
- **在** = 位置、進行中（At / In）

### 7. 智能語境判斷（哪 / 那）

| 錯誤用法 | 正確用法 | 說明 |
|---------|---------|------|
| 那裡是？ | 哪裡是？ | 疑問語境用「哪」 |
| 哪裡很漂亮 | 那裡很漂亮 | 指示語境用「那」 |

**判斷原則：**
- **哪** = 疑問詞（where/which）- 用於問句、疑問語境
- **那** = 指示詞（that）- 用於陳述、指示語境

**語境偵測**：自動檢測問句特徵（嗎、呢、什麼等）決定使用「哪」或「那」

## 安裝方式

### 開發者模式安裝

1. 開啟 Chrome 瀏覽器
2. 前往 `chrome://extensions/`
3. 開啟右上角「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇 `AtAgain` 資料夾
6. 完成！

## 使用方式

安裝後自動運作，在任何網頁的輸入框打字時：
- 輸入空格、標點或離開輸入框時自動修正
- 不打斷打字流程，即時修正

## 技術說明

- 使用 Chrome Extension Manifest V3
- 內容腳本（Content Script）注入所有網頁
- 監聽 `input`、`keyup`、`blur` 事件
- 支援 `<input>`、`<textarea>` 和 `contenteditable` 元素
- 修正執行順序：`direct` → `regexDe`（批次 regex）→ `lookupDe/Quan/Yi/Ji/Zuo`（詞組表）→ `contextual`（語境判斷）
- `_isProcessing` flag 防止 dispatchEvent 造成無限遞迴
- 詞組表以 key 長度降序排序，避免短字串覆蓋長字串

## 隱私政策

本擴充功能**完全本機運作**，不收集、不儲存、不傳送任何使用者資料。詳見 [privacy.html](privacy.html)。

- 不申請任何特殊許可權（permissions: []）
- 不連線任何外部伺服器
- 不使用 Google Analytics 或任何追蹤工具
