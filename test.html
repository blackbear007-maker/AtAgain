🎯 2. 「的/得」對照表測試（300+組）

動詞後用「得」，形容詞後用「的」

→ 跑得快
的→得

的→得

→ 吃得好

的→得

→ 聽得懂
的→得
看的見
→ 看得見

⏰ 4. 「已/以」對照表測試（300+組）

🤔 6. 「在/再」語境判斷測試

「再」= 重複/再次，「在」= 位置/進行中

在→再
在見
→ 再見
在→再
在來
→ 再來
再→在
再家
→ 在家
再→在
現再
→ 現在
再→在
正再吃飯
→ 正在吃飯

❓ 7. 「哪/那」語境判斷測試

「哪」= 疑問，「那」= 指示

那→哪
那裡是？
→ 哪裡是？
那→哪
那個人嗎？
→ 哪個人嗎？
哪→那
哪裡很漂亮
→ 那裡很漂亮
